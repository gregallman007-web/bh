# Backhouse Solicitors — Homepage Redesign Concepts

Four working HTML homepage concepts for backhouse-solicitors.co.uk, plus an index page linking them.

## Contents
- `index.html` — concept picker (start here)
- `Backhouse Homepage.dc.html` — Option 1: The Evolution (refined navy/amber)
- `Backhouse Homepage v2.dc.html` — Option 2: The Editorial (charcoal-teal design system)
- `Backhouse Homepage v3.dc.html` — Option 3: The Modern (rounded, mosaic grid)
- `Backhouse Homepage v4.dc.html` — Option 4: The Modern + typewriter hero
- `support.js` — page runtime (required, keep alongside the HTML files)
- `assets/`, `uploads/` — logos and imagery

## Deploy
Serve the folder as-is (GitHub Pages: point Pages at this directory). No build step required.
